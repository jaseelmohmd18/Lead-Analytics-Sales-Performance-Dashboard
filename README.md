1. Project Title

Lead Analytics & Sales Performance Dashboard

2. Short Description

A Power BI dashboard project designed to analyze lead generation, sales performance, and conversion metrics. The dashboard provides interactive visualizations and KPI tracking to support data-driven business decisions

3. Tech Stack

Power BI
Power Query
DAX
Data Modeling
Excel

4. Data Source

The dataset contains lead generation and sales data collected from various marketing campaigns and sales channels. It includes information on lead sources, campaigns, sales stages, conversion status, and performance metrics.

5. Features

Data Cleaning and Transformation using Power Query
Data Modeling and Relationship Management
KPI Development using DAX
Lead Conversion Analysis
Sales Performance Monitoring
Interactive Filters and Slicers
Drill-through Reports
Bookmarks and Tooltips
Dynamic Dashboard Navigation

6. Dashboard KPIs

Total Leads
Converted Leads
Conversion Rate
Sales Performance
Campaign Performance
Lead Source Analysis

7. Business Impact

Reduced manual reporting effort through dashboard automation.
Improved visibility into lead conversion and sales performance.
Enabled real-time monitoring of business KPIs.
Supported data-driven sales and marketing decisions.
